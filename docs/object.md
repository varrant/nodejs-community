# `object` 模型

适用于：文字、问题、帮助等展示性资源。

`*`表示虚拟字段

- `author` 作者
- `title` 标题
- `uri` URI
- `section` 板块
- `category` 分类
- `column` 专栏
- `labels` 标签
- `introduction` markdown 简介
- `introductionHTML` html 简介
- `content` markdown 内容
- `contentHTML` html 内容
- `publishAt` 发布时间
- `updateAt` 最近修改时间
- `updateList` 修改历史
- `isDisplay` 是否前台可见
- `isSticky` 是否被置顶
- `isRecommend` 是否被推荐
- `isEssence` 是否被精华
- `color` 加色
- `score` 被加分，默认系统添加1分
- `scoreList` 被加分历史
- `viewByCount` 被阅读次数
- `commentByCount` 被评论次数
- `replyByCount` 被回复次数
- `favoriteByCount` 被收藏次数
- `applyByCount` 被申请次数
- `hasAccepted` 是否被接受了
- `hasCertificated` 是否被认证了
- `contributors` 最新的 5 位贡献者
- `meta` 元信息
	- `url` 组织链接，通常为官网
	- `company` 组织所在企业
	- `location` 组织所在地
	- `memberCount` 组织成员数量
	- `time` 组织创立时间
