# 全局变量
- window['-csrf-'] 请求令牌
- window['-engineer-'] 当前用户信息
- _window['-refresh-'] 登录之后是否刷新【部分页面】_
- window['-section-'] 页面所在版块ID【部分页面】
- window['-id-'] 详情页ID【部分页面】
- window['-page-'] 分页值【部分页面】
- window['-author-'] 作者信息【部分页面】


# HTTP 状态值
- 200 正常
- 301 跳转
- 400 未验证
- 401 未登陆
- 403 无权限
- 404 未找到
- 406 验证失败
- 500 服务错误
